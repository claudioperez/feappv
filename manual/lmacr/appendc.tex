\chapter[Solution Command Manual]{Solution Command Manual Pages}
\label{appxc}

{\sl FEAPpv} has several options which may be used to solve problems.
The solution strategy is based on a {\it command language} approach in
which users write each step using the available commands.  The following
pages summarize the commands currently available in {\sl FEAPpv}.  These
include optioins needed to solve most problems; however,
provisions are also available for users to include their
own solution routines through use of {\tt UMACRn} subprograms.  Methods
to write and interface user routines to the program are described in the
{\sl FEAP} Programmers Manual.
